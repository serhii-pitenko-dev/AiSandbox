### Map
- Grid of Cells
- Incapsulate map general information and list of cells with objects
- Can move objects between cells
- In the domain map coordinates X and Y started from 0 at the bottom-left corner.

### Map Objects
- Enemy, Hero, Block
- Incapsulate behavior and state

### Playground
- Implements pattern "Aggregate root"
- Incapsulate Map, Map Objects and manage Map Objects behavior execution
